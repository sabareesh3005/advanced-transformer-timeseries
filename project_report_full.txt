TITLE: Multivariate Time-Series Forecasting using LSTM + Attention, Hyperparameter Optimization, and ARIMA Baseline Comparison

1. DATASET
----------
A synthetic multivariate time‑series dataset was used, containing 6 correlated features and multiple patterns:
- Seasonal patterns (weekly and yearly cycles)
- Long-term trend
- Short-term noise
- Feature‑level interactions

The dataset contains 300+ time steps (placeholder), stored in:
    multivariate_series.csv

Target variable selected: feat_1.

2. MODEL ARCHITECTURE (LSTM + ATTENTION)
----------------------------------------
The deep learning model is based on:
- LSTM encoder (captures temporal dependencies)
- Attention mechanism (learns which time steps are important)
- Fully connected decoder for forecasting a 14‑step horizon

Attention mechanism allows interpretability by assigning weights to recent past time steps.

3. HYPERPARAMETER OPTIMIZATION (OPTUNA)
---------------------------------------
Optuna was used to tune:
- hidden dimension size
- learning rate
- dropout rate
- number of LSTM layers
- batch size

Example best parameters (placeholder):
    hidden_dim = 64
